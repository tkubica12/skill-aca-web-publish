# ACA Web Publish Skill

Opinionated Agent Skill for deploying private static HTML/media sites on Azure Container Apps with app-level authentication and private Blob Storage.

## Install

```bash
gh skill install tkubica12/skill-aca-web-publish aca-web-publish --scope user
```

## Preview

```bash
gh skill preview tkubica12/skill-aca-web-publish aca-web-publish
```

## Skill layout

The published skill follows the Agent Skills convention:

```text
skills/aca-web-publish/SKILL.md
skills/aca-web-publish/references/
skills/aca-web-publish/assets/
```
